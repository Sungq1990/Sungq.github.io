MySQL事务隔离级别

隔离级别 脏读 不可重复读 幻读
未提交读 可能 可能 可能
已提交读 不可能 可能 可能
可重复读 不可能 不可能 可能
可串行化 不可能 不可能 不可能

Read Uncommit：未提交读。允许脏读。
Read Commit ： 提交读。只能读到其他事务已提交的内容。允许不重复读。
Repeated Read ： 可重复读。同一个事务内的查询与事务开始时是一致的。允许幻读。Mysql默认的级别。
Serializable ： 串行化的读。每次读都要获取表级锁，读写互相阻塞
